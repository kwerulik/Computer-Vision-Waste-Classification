

# Computer-Vision-Waste-Classification
This project is a computer vision assignment developed for the Artificial Intelligence Fundamentals course at the University of Gdańsk. The primary objective is to build and evaluate a deep learning model capable of accurately classifying waste images to support automated recycling processes.
